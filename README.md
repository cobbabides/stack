# The Stack

Personal supplement tracking PWA for daily stack management, Fadiman microdose protocol, and body tracking.

## Features

- **Today** — Daily supplement checklist with Fadiman day type auto-detection, mood + symptom logging
- **Go Out** — Situational stack guide (social, deep work, training, meetings, stress)
- **Track** — Weight log with trend chart, food log, supplement history
- **Stack** — Full supplement list with brand selector, pros/cons, and monthly cost estimate
- **Protocol** — Fadiman calendar, cycle management, non-negotiable rules

## Stack Includes

Brain · Body · Gut · BP Management · Piracetam · Aniracetam · Stamets Stack

## Setup

1. Open in Safari on iPhone
2. Tap Share → Add to Home Screen
3. Set your Fadiman start date in the Protocol tab
